Capabilities

Everything a Tech Needs, In One EXE

Seven major feature panels covering the full lifecycle of PC setup, maintenance, migration, and recovery.

🔄

Backup & Restore

Step-by-step wizards with smart filtering, AES-256 encryption, NTFS permission handling, plus integrated screenshot settings and BitLocker key backup.

AES-256 NTFS BitLocker Wizards
📊

Diagnostics & Benchmarking

SMART disk health, CPU/disk/memory benchmarks, stress testing, thermal monitoring, system maintenance tools, and HTML/JSON/CSV reports.

SMART Benchmarks Stress Test Reports
💿

Image Flasher

Flash disk images to USB — supports ISO, IMG, WIC, ZIP, GZ, BZ2, XZ, DMG, and DSK. Download Windows ISOs via UUP Dump API or Linux ISOs from a built-in distro catalog. Full UEFI boot support.

ISO IMG UEFI Windows Linux 9 Formats
📁

File Management

Organize files by extension, find duplicates with parallel hashing, and grep-style content search with regex, metadata, archive, and binary/hex modes.

Regex Dedup EXIF Hex Search
📦

Application Management

Scan installed apps, match to Chocolatey packages, bulk-install on new PCs. Offline deployment with direct installer downloads. Deep uninstall with leftover scanning.

Chocolatey Offline Deploy Deep Uninstall Registry Diff
🌐

Network Management

10-tool diagnostic suite: ping, traceroute, MTR, DNS, port scan, bandwidth, WiFi analysis. Adapter inspector with backup/restore and WiFi QR code manager.

Ping Traceroute Port Scan WiFi QR
📧

Email Tools

Browse PST, OST, and MBOX email archives without Outlook. Search, export (EML/CSV/VCF/ICS), contacts, calendar views, and attachments browser. Includes multi-threaded OST/PST converter with 8 output formats and IMAP cloud upload.

PST OST MBOX No Outlook OST Converter IMAP Upload
Migration

Backup & Restore That Just Works

Guided wizards walk you through every step of user-profile migration. Smart filtering picks the data that matters and leaves the rest behind.

  • AES-256 encryption for sensitive data
  • NTFS permission preservation and restoration
  • Screenshot Settings backup (Windows capture prefs)
  • BitLocker recovery key export
  • Selective folder inclusion with preview
Step-by-step wizard progress
Deployment

App Installation & Offline Deploy

Scan a source PC, match every app to Chocolatey, then bulk-install the entire stack on a fresh machine — even without internet.

  • Registry scan with Chocolatey package matching
  • Confidence-tiered matching (high, medium, low, manual)
  • Build Bundle: self-contained offline nupkg packages
  • Direct Download: grab raw EXE/MSI installers
  • Preset lists: Office PC, Developer, Media, and more
  • Deep uninstall with leftover scanning and registry diffs
Package installation progress

Complete Network Diagnostic Suite

Ten diagnostic tools in one panel. Adapter configuration backup/restore. WiFi credential management with QR code generation.

  • Ping, Traceroute, MTR path analysis
  • DNS lookup, Port scanner, Bandwidth tests

  • Firewall auditor and Network share browser
  • Adapter config backup/restore to JSON
  • WiFi QR codes — single and batch generation
